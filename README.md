# Voice Shopping Assistant

A simple React-based shopping list app that supports **voice commands**, **manual item entry**, **quantity editing**, and **smart suggestions**.

## Features
- Add/remove items using voice commands  
- Manual item creation with quantity input  
- Increase/decrease item quantity  
- Search items  
- Smart seasonal & recommendation suggestions  
- Optional product details (battery, memory, processor)  
- Multilingual speech recognition
